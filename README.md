## 📌 Overview
This repository contains the interactive supplementary digital asset for the manuscript: **"Democratizing Precision Brain Surgery: Integrating Connectomic Priors and Real-Time BCI Telemetry for Scalable Human Augmentation."** This Software-as-a-Medical-Device (SaMD) prototype simulates a paradigm-shifting framework for neuro-oncological surgery. By fusing pre-operative structural connectomics (DTI/fMRI) with real-time electrophysiological telemetry, this system acts as a computational cognitive multiplier for human neurosurgeons. It circumvents the mechanical latencies and prohibitive capital costs of autonomous robotic suites, dynamically compensating for intraoperative brain shift and electromagnetic interference (EMI) to achieve sub-millimeter tissue preservation.

## ⚙️ Core Subsystems
The master engine features five interactive modules accessible via the navigation hub, rigorously aligned with established biophysical neuro-monitoring principles:

1. **Comparative Surgical Framework:** A health-economics and structural modeling tool comparing BCI-Human augmentation against autonomous robotic platforms under varying degrees of brain shift and resource availability.
2. **Connectome Signal Diaschisis Graph:** A dynamic global network visualizer monitoring cortico-cortical evoked potentials (CCEPs). It demonstrates how focal surgical stunning at a primary node (M1) triggers systemic hodotopical diaschisis, automatically prompting a critical resection halt rather than assuming instantaneous neuroplasticity.
3. **Triad Architecture Simulator:** A systems-flow model demonstrating the sub-20ms latency closed-loop pipeline from pre-op prior ingestion to the surgeon's heads-up display (HUD).
4. **Subcortical Mapping Engine (DES):** A real-time spatial pathfinding simulator utilizing Active Direct Electrical Stimulation (DES). It mathematically links monopolar stimulation thresholds (mA) to motor evoked potential (MEP) latency, applying the 1mA = 1mm biophysical rule to prevent eloquent white matter tract transection.
5. **Full System Dashboard:** The integrated master UI for simultaneous multi-parameter testing, featuring Adaptive Common Average Referencing (ACAR) and Independent Component Analysis (ICA) algorithms to scrub high-voltage diathermy (EMI) noise from physiological data feeds.
