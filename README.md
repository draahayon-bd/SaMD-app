## 📌 Overview
This repository contains the interactive supplementary digital asset for the manuscript: **"Democratizing Precision Brain Surgery: Integrating Connectomic Priors and Real-Time BCI Telemetry for Scalable Human Augmentation."** This Software-as-a-Medical-Device (SaMD) prototype simulates a paradigm-shifting framework for neuro-oncological surgery. By fusing pre-operative structural connectomics (DTI/fMRI) with real-time electrophysiological telemetry (high-gamma ECoG), this system acts as a computational cognitive multiplier for human neurosurgeons. It circumvents the mechanical latencies and prohibitive capital costs of autonomous robotic suites, dynamically compensating for intraoperative brain shift and electromagnetic interference (EMI) to achieve sub-millimeter tissue preservation.

## ⚙️ Core Subsystems
The master engine features five interactive modules accessible via the navigation hub:

1. **Comparative Surgical Framework:** A health-economics and structural modeling tool comparing BCI-Human augmentation against autonomous robotic platforms under varying degrees of brain shift and resource availability.
2. **Connectome-Weighted Signal Rerouting:** A dynamic graph-theory visualizer demonstrating the automated redistribution of feature-extraction weights across secondary cortical nodes (e.g., SMA, PMC) when a primary node (M1) is compromised by surgical stunning.
3. **Triad Architecture Simulator:** A systems-flow model demonstrating the sub-20ms latency closed-loop pipeline from pre-op prior ingestion to the surgeon's heads-up display (HUD).
4. **Intraoperative Mapping Engine:** A real-time spatial pathfinding simulator showcasing spatiotemporal artifact scrubbing of diathermy (EMI) noise from high-gamma (70-150 Hz) signal feeds.
5. **Full System Dashboard:** The integrated master UI for simultaneous multi-parameter testing.
